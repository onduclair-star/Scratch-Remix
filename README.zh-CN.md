# Scratch-Remix
厌倦了原版 Scratch？积木功能不够用？帧率卡顿掉帧？调试过程枯燥得让人抓狂？

那就来试试 **Scratch-Remix** 吧 —— 这是一款由独立开发者基于 Unity 引擎全新打造的免费开源 Scratch 重构版本！

---

## 功能亮点

### 增强型积木系统
- **新增强力积木**：例如用于实现丝滑动画的 `Time.deltaTime` 积木。更多控制类、事件类和运算类积木正在开发中！
- **支持自定义积木**：对现有积木不满意？欢迎提交创意想法，或者直接自行添加新积木！
- **[开发中] 优化版积木解释器**：全面支持复杂逻辑运算（条件判断、循环、嵌套条件等）。

### 现代化流畅交互界面
- 顺滑的拖拽操作体验 + 智能自动隐藏工具栏 —— 兼具现代设计感与儿童友好属性。
- **[开发中] 内置角色编辑器**：支持角色拖拽、多帧动画创建（功能处于积极开发阶段）。

### 卓越性能表现
- 基于 Unity 2D 引擎开发，即使在低配设备上也能稳定实现 60 帧满帧运行。
- **正式版后计划**：推出 Unity 插件版本 —— 让完整的 Unity 引擎为你处理高性能运算需求！
