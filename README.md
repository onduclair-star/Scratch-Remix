## Features

### Enhanced Block System
- New powerful blocks: e.g., `Time.deltaTime` for buttery-smooth animations. More control, events, and operators on the way!
- Custom blocks supported: Not happy with what's there? Submit ideas or add your own!
- [WIP] Optimized block interpreter with full support for complex logic (if/else, loops, nested conditions, etc.).

### Modern & Smooth UI
- Silky drag-and-drop experience + smart auto-hiding toolbar ‚Äî feels modern but stays kid-friendly.
- [WIP] Built-in sprite editor: drag sprites, create multi-frame animations (actively in development).

### Blazing Performance
- Powered by Unity 2D, targeting rock-solid 60 FPS even on low-end hardware.
- Post-MVP: Release a Unity package version ‚Äî let the full Unity engine handle the heavy lifting!
